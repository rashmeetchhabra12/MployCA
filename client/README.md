# Angular Client

Angular front end for the role-based user dashboard.

## Local Development

```bash
npm install
npm start
```

The development server runs on `http://localhost:4200` and expects the API to be available at `http://localhost:4000/api`.

## Build

```bash
npm run build
```

The production output is written to `dist/client/browser` and can be served by the Express API after the root build command is run.
